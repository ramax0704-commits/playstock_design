## 개요

- **사용자 목표**: 투자 포트폴리오 확인, 보유 자산 관리, 프로필 정보 조회
- **인증 필요**: 예 (로그인된 사용자만)

## 레이아웃

### 구조
- **상단 고정 헤더**: 상태바
- **프로필 섹션**: 사용자 정보 및 프로필 진입
- **바디 섹션** (스크롤):
  - 보유 자산 요약 카드 (2개 이상)
  - 추가 정보 섹션들
- **하단 고정 네비게이션**: 3개 탭 (홈, 찜, 마이페이지)

### 뷰포트
- **폭**: 375px (모바일)
- **높이**: 812px (바디 스크롤 가능)

### 그리드 & 간격
- **상태바**: 44px
- **헤더**: 56px
- **프로필 섹션 패딩**: 0px (상단), 20px (좌우)
- **바디 패딩**: 20px (좌우), 16px (상하)
- **네비게이션 높이**: 80px (하단 고정)
- **섹션 간격**: 20px

## 컴포넌트

### 1. 상태 바
- **높이**: 44px
- **배경색**: `$bg-secondary` (#F7F8FA)
- **콘텐츠**: 시간, WiFi/배터리 아이콘

### 2. 헤더
- **높이**: 56px
- **배경색**: `$bg-secondary`
- **배치**: 우측 정렬 (`justify-end`)
- **콘텐츠**: 우측 아이콘 그룹 (설정 등)
  - 아이콘 크기: 24x24px
  - 갭: 16px

### 3. 프로필 섹션
- **배경색**: `$bg-secondary`
- **패딩**: 0px (상단), 20px (좌우)
- **구성**:
  - 프로필 텍스트
  - "우측 chevron-right 아이콘" → 프로필 관리 페이지로 이동
- **상호작용**: 클릭 시 프로필 상세 페이지로 이동

### 4. 바디 (Body)

#### 4-1. 보유 자산 요약 카드 (여러 개)
- **개수**: 2개 이상
- **구성**:
  - 카드 타이틀 (예: "보유 중인 조각")
  - 자산 정보 (채널명, 보유 수량, 현재가 등)
  - 프로필 이미지 (원형)
  - 상세 정보 링크 또는 버튼
- **배경색**: `#ffffffff` (white)
- **보더**: 1px `$border`
- **모서리**: radius-lg (12px)
- **패딩**: 20px
- **갭**: 16px (요소 간)
- **섀도우**: blur 6px, color #00000026 (경미한 섀도우)

#### 4-2. 추가 섹션들
- 배당금 정보
- 수익 현황
- 기타 투자 정보

### 5. 하단 네비게이션 (Bottom Tab Bar)
- **높이**: 80px (고정)
- **배경색**: `$bg-primary` (#ffffff)
- **상단 보더**: 1px `$border`
- **패딩**: 20px (상하), 12px (좌우)
- **콘텐츠**: 3개 탭
  - 홈 (아이콘: house): 현재 페이지와 다른 탭
    - 비활성: `$text-tertiary`
  - 찜 (아이콘: heart): 비활성
    - 비활성: `$text-tertiary`
  - 마이페이지 (아이콘: user): 현재 페이지 (활성)
    - 활성: `$text-primary`
- **탭 레이아웃**: 수평 중앙 정렬, space-around
- **각 탭 구성**:
  - 아이콘: 24x24px
  - 텍스트: Pretendard Variable, 11px, regular
  - 아이콘-텍스트 갭: 4px
  - 아이콘-텍스트 정렬: 수직 중앙

## 디자인 토큰

### 색상
| 토큰 | 값 | 용도 |
|---|---|---|
| `$bg-secondary` | #F7F8FA | 헤더, 프로필 배경 |
| `$bg-primary` | #FFFFFF | 네비게이션 배경 |
| `surface` | #FFFFFF | 카드 배경 |
| `$border` | #EAECEF | 보더, 네비게이션 상단선 |
| `$text-primary` | #1F1F1F | 활성 탭 텍스트 |
| `$text-tertiary` | #9CA3AF | 비활성 탭 텍스트/아이콘 |

### 타이포그래피
| 역할 | 폰트 | 크기 | 굵기 |
|---|---|---|---|
| 카드 타이틀 | Pretendard Variable | 14px | 400 (regular) |
| 탭 텍스트 | Pretendard Variable | 11px | 400 (regular) |

### 간격
| 토큰 | 값 | 용도 |
|---|---|---|
| 기본 패딩 | 20px | 섹션 좌우 패딩 |
| 바디 패딩 | 20px (좌우), 16px (상하) | 바디 섹션 |
| 섹션 간격 | 20px | 카드 간 간격 |
| 카드 갭 | 16px | 카드 내부 요소 간 |
