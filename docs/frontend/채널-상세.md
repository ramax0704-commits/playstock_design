# 채널 상세 (SCR-I002)

> 사용자가 특정 채널의 상세 정보를 탭별로 탐색할 수 있는 페이지입니다. 조각정보, 가치평가, 수익 구조 세 개의 탭을 통해 채널의 다양한 정보를 제공합니다.

**관련 UI 페이지**: 
- SCR-I002 채널 상세 / 조각정보
- SCR-I002 채널 상세 / 가치평가
- SCR-I002 채널 상세 / 수익 구조

## 개요

- **라우트**: `/channels/:id` (채널 상세)
- **사용자 목표**: 투자하려는 채널의 상세 정보 조회 및 검토
- **인증 필요**: 예

---

## 레이아웃

### 구조
- **상단 고정 헤더**: 뒤로가기 버튼, 채널명 (뜬뜬 DdeunDdeun), 찜하기 아이콘
- **채널 이미지**: 원형 이미지 + 채널 정보 (브랜드, 설명 등)
- **탭 네비게이션**: 조각정보 / 가치평가 / 수익 구조 (4개 탭 중 3개 활성)
- **동적 콘텐츠 영역**: 선택된 탭에 따라 다른 정보 표시
- **하단 고정 CTA**: "조각 구매" 버튼 (또는 "구매하기")

### 뷰포트
- **폭**: 375px (모바일)
- **높이**:
  - 조각정보 탭: 1300px
  - 가치평가 탭: 1085px
  - 수익 구조 탭: 926px

### 그리드 & 간격
- **헤더 높이**: 44px (상태바) + 56px (탑 헤더)
- **탭 높이**: 약 50px
- **패딩**: 20px (좌우)
- **섹션 간격**: 20px

## 컴포넌트

### 1. 상태 바 (Status Bar)
- **높이**: 44px
- **콘텐츠**: 시간 (9:41), WiFi/배터리 아이콘
- **배경색**: `#ffffff`

### 2. 헤더 (Top Header)
- **높이**: 56px
- **콘텐츠**:
  - 좌측: 뒤로가기 아이콘 (arrow-left, 24x24px)
  - 중앙: 채널명 "뜬뜬 DdeunDdeun" (Pretendard Variable, 16px, weight-500)
  - 우측: 찜하기 아이콘 (heart, 24x24px, `$text-tertiary`)
- **배경색**: `#ffffff`
- **패딩**: 20px (좌우), 12px (상하)

### 3. 채널 이미지 & 정보 (Profile Section)
- **이미지**: 원형, 44x44px, 배경 #CCCCCC
- **정보**: 브랜드명, 설명 등을 수직 스택으로 표시
- **레이아웃**: 이미지 좌측 정렬, 정보 우측

### 4. 탭 네비게이션 (Tab Group)
- **콘텐츠**: 4개 탭
  1. 조각정보 (tap/on 또는 tap/off)
  2. 가치평가 (tap/on 또는 tap/off)
  3. 수익 구조 (tap/on 또는 tap/off)
  4. 추가 탭 (tap/off)
- **스타일**: 각 탭 아이콘 + 텍스트 (아이콘 상단, 텍스트 하단)
- **간격**: 탭 사이 30px
- **하단 보더**: 1px `$border`

### 5. 콘텐츠 영역 (탭별 상이함)

#### 5-1. 조각정보 탭
- **구성**:
  1. 조각 판매 카드 (sale-info frame)
     - 원형 이미지 + 채널 정보
     - 판매 진행률 (saleProgress)
     - 가격 정보 (가격대, 남은 조각 수 등)
  2. 조각 잔디 섹션 (section with title + 이미지 리스트)
  3. 기타 정보

#### 5-2. 가치평가 탭
- **구성**:
  1. AI 가치평가 카드 (그래디언트 배경, 파란색 계열)
     - 평가 점수, 차트 등
  2. 카테고리 비교 카드 (하양 배경)
     - 이 채널 vs 다른 채널 비교

#### 5-3. 수익 구조 탭
- **구성**:
  1. 섹션 타이틀: "채널 수익 구조"
  2. 예상 월 수익 카드 (white, shadow 포함)
  3. 상세 항목 리스트
  4. 설명 텍스트 (면책조항)

### 6. 하단 CTA (Bottom CTA)
- **높이**: 96px
- **버튼**: `btn/defult` (기본 버튼, fill: `$primary`)
- **텍스트**: "조각 구매" 또는 "구매하기"
- **배경색**: `$bg-primary` (#ffffff)
- **상단 보더**: 1px `$border`

## 디자인 토큰

### 색상
| 토큰 | 값 | 용도 |
|---|---|---|
| `$bg-primary` | #FFFFFF | 메인 배경, 헤더, CTA 배경 |
| `$surface` | #FFFFFF | 카드 배경 |
| `$border` | #EAECEF | 보더 라인 |
| `$text-primary` | #1F1F1F | 제목, 헤더 텍스트 |
| `$text-tertiary` | #9CA3AF | 비활성 아이콘 (찜 아이콘) |
| `$primary` | #FF5252 | CTA 버튼 배경 |
| Gradient (가치평가) | #1f69ff → #95b8ff | AI 가치평가 카드 배경 |

### 타이포그래피
| 역할 | 폰트 | 크기 | 굵기 |
|---|---|---|---|
| 시간 표시 | Pretendard Variable | 14px | 600 (semibold) |
| 채널명 | Pretendard Variable | 16px | 500 (medium) |
| 섹션 타이틀 | Pretendard Variable | 14px | 700 (bold) |
| 설명 텍스트 | Pretendard Variable | 11px | 400 (regular) |

### 간격
| 토큰 | 값 | 용도 |
|---|---|---|
| 기본 패딩 | 20px | 섹션 좌우 패딩 |
| 섹션 간격 | 20px | 섹션 간 수직 간격 |
| 탭 간격 | 30px | 탭 아이콘 간 간격 |
| 카드 패딩 | 20px | 카드 내부 패딩 |
