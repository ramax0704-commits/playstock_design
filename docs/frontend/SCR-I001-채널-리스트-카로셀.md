# SCR-I001 채널 리스트 (카로셀)

> 투자자가 현재 거래 중인 채널을 둘러보고, 카테고리별로 필터링하여 투자할 채널을 찾는 화면입니다.

---

## 디자인 캡처

### 기본 화면
![SCR-I001 채널 리스트 카로셀 기본 화면](assets/BaQBk.png)

---

## 개요

- **라우트**: `/channels` 또는 `/home` (투자자 홈)
- **사용자 목표**: 현재 거래 중인 채널 목록을 보고, 관심 채널을 찾아 투자하기
- **인증 필요**: 예 (로그인 후 접근 가능)
- **뷰포트**: 375px × 1681px (모바일 앱)

---

## 레이아웃

### 구조

**전체 레이아웃**: 고정 헤더 + 스크롤 콘텐츠 + 고정 바텀 탭바

1. **상태바** (44px)
   - 시간(9:41), WiFi 아이콘, 배터리 아이콘
   - 배경색: `$bg-secondary` (#F7F8FA)

2. **탑 헤더** (56px)
   - 좌측: 브랜드명 (20px, weight: 700)
   - 우측: 검색 아이콘, 알림 아이콘

3. **히어로 이미지** (162px)
   - 현재 거래 중인 채널의 대표 이미지 또는 배너
   - 그레이 플레이스홀더 (#CCCCCC)

4. **인기 채널 섹션**
   - 섹션 제목: "지금 거래중인 인기 채널" (20px, weight: 700)
   - **카로셀 (수평 스크롤)**
     - `channel card` 컴포넌트 × 2개 (보이는 영역)
     - 카드 폭: 335px (화면 너비 375px - 양쪽 padding 40px)
     - 카드 간격: 16px
     - `clip: true` 로 좌우 스크롤 활성화

5. **카테고리 필터 탭**
   - 섹션 제목: "채널"
   - 탭 목록: "배당 비율", "평가 가치", ... (3개 더)
   - 액티브 탭 하단에 2px 언더라인

6. **채널 리스트 (수직)**
   - `channel list` 컴포넌트 × 4개
   - 간격: 16px
   - 각 채널 카드는 채널명, 구독자 수, 투자 금액, 배당 비율, 남은 조각 수 표시

7. **추가 버튼**
   - "더 보기" 버튼 (Ghost 스타일)
   - 가운데 정렬

8. **바텀 탭바** (80px, 고정)
   - 4개 탭: 홈(활성), 검색, 찜, 마이
   - 홈 탭은 아이콘/텍스트 모두 검정색
   - 나머지 탭은 그레이색 (#9CA3AF)
   - 상단 보더: 1px (#EAECEF)

### 그리드 & 간격

```
전체 폭: 375px
패딩: 20px (상하좌우, 탭바 제외)
섹션 간격: 20-24px
카드 간격: 16px (horizontal), 10-20px (vertical)
```

**타입그래피 간격**:
- 제목: 20px, 700 weight (Pretendard Variable)
- 섹션: 16px, 600 weight
- 본문: 14-16px, 400 weight
- 캡션: 11-12px, 400 weight

│

│  └─ 메타 정보 (stats)

## 인터랙션 & 화면 흐름

```
[페이지 로드] 
  → 인기 채널 카로셀 (자동 스크롤 or 수동 드래그 가능)

[카로셀 카드 "구매하기" 클릭]
  → SCR-I003 거래 패널 (Transaction Panel) 이동

[카테고리 탭 클릭]
  → 필터 적용 (예: "배당 비율" 기준 정렬)
  → 채널 리스트 재정렬

[채널 리스트 항목 클릭]
  → SCR-I002 채널 상세 (Channel Detail) 이동

["더 보기" 버튼 클릭]
  → 추가 채널 로드 (또는 전체 목록 페이지 이동)

[바텀 탭 클릭]
  → "홈" → 현재 페이지 (이미 활성)
  → "검색" → 검색 페이지 이동
  → "찜" → 찜한 채널 목록 이동
  → "마이" → 투자자 마이페이지 (SCR-I007) 이동
```

## 개발자 참고사항

1. **카로셀 구현**
   - `clip: true` 속성으로 좌우 오버플로우 숨김
   - 네이티브 스크롤 또는 라이브러리 (Swiper, react-scroll 등) 활용
   - 자동 스크롤 여부는 <!-- TODO: 디자이너 확인 필요 -->

2. **카테고리 필터 탭**
   - 현재 "배당 비율", "평가 가치", 그리고 2개 탭 더
   - 정확한 탭 라벨은 <!-- TODO: PM/기획팀 확인 필요 -->
   - 탭 스크롤 필요 시 수평 스크롤 구현

3. **채널 리스트**
   - 처음 4개만 표시, "더 보기" 클릭 시 추가 로드
   - 페이지네이션 or 무한 스크롤 구현 필요 (<!-- TODO: 확인 필요 -->)

4. **비활성 상태**
   - 로그인하지 않은 사용자는 이 화면 접근 불가 (SCR-CM002 로그인 페이지로 리다이렉트)

5. **이미지 로딩**
   - 채널 썸네일, 히어로 이미지는 플레이스홀더 표시 후 로드
   - 로딩 실패 시 회색 플레이스홀더 유지

6. **아이콘 패밀리**
   - lucide 폰트 사용 (house, search, heart, user, wifi, battery, bell)
   - 모바일 앱에서 lucide 또는 해당 플랫폼의 시스템 아이콘 대체 가능

7. **D-day 태그**
   - 카로셀 카드에서 "D-10", "D-3" 등 판매 마감일 표시
   - 동적으로 계산되어야 함 (현재 날짜 - 마감일)

8. **접근성**
   - 바텀 탭바는 각 탭이 최소 44×44px (터치 타겟 크기) 이상
   - 모든 아이콘에 ARIA label 추가
   - 색상만으로 정보 전달하지 않기 (예: 배당 비율 "%"는 필수)

9. **성능**
   - 리스트 아이템 많을 시 가상 스크롤(virtualization) 고려
   - 이미지 lazy-loading 구현

10. **데이터 바인딩**
    - 각 채널 카드/리스트는 백엔드 API에서 받은 데이터로 동적 렌더링
    - 필수 필드: `channelId`, `channelName`, `subscribers`, `thumbnail`, `pricePerFragment`, `dividendRatio`, `soldFragments`, `totalFragments`, `daysUntilSaleEnd`
