# 투자자 포트폴리오

> 사용자가 자신의 투자 현황, 보유 조각, 수익 정보를 조회할 수 있는 대시보드 기능입니다.

**관련 UI 페이지**: SCR-I007 투자자 마이페이지

---

## 데이터 모델

### UserProfile (사용자 프로필)

| 필드 | 타입 | 필수 여부 | 설명 |
|---|---|---|---|
| `id` | UUID | 필수 | 사용자 ID |
| `email` | string | 필수 | 이메일 |
| `name` | string | 필수 | 사용자명 |
| `profileImageUrl` | string | 선택 | 프로필 이미지 URL |
| `createdAt` | Date | 필수 | 가입 날짜 |

### Portfolio (포트폴리오 요약)

| 필드 | 타입 | 필수 여부 | 설명 |
|---|---|---|---|
| `userId` | UUID | 필수 | 사용자 ID (외래키) |
| `totalAssets` | number | 필수 | 총 자산 (보유 조각 총 가치) |
| `totalDividends` | number | 필수 | 누적 배당금 |
| `totalHoldings` | number | 필수 | 총 보유 조각 수 |
| `totalChannels` | number | 필수 | 투자 중인 채널 수 |

### Holding (보유 자산)

| 필드 | 타입 | 필수 여부 | 설명 |
|---|---|---|---|
| `id` | UUID | 필수 | 보유 기록 ID |
| `userId` | UUID | 필수 | 사용자 ID (외래키) |
| `channelId` | UUID | 필수 | 채널 ID (외래키) |
| `quantity` | number | 필수 | 보유 조각 수 |
| `purchasePrice` | number | 필수 | 구매 당시 총 가격 |
| `currentValue` | number | 필수 | 현재 가치 |
| `gainLoss` | number | 필수 | 수익/손실 = currentValue - purchasePrice |
| `gainLossPercent` | number | 필수 | 수익률 (%) = (gainLoss / purchasePrice) × 100 |
| `accumulatedDividends` | number | 필수 | 누적 배당금 |
| `purchasedAt` | Date | 필수 | 구매 날짜 |

### 사용자 프로필 조회
```
GET /api/investor/profile
```

**인증 필요**: 예

**응답** (성공 - 200):
```json
{
  "data": {
    "id": "user_uuid",
    "email": "user@example.com",
    "name": "김지은",
    "profileImageUrl": "https://...",
    "createdAt": "2025-01-01T00:00:00Z"
  }
}
```

### 보유 자산 상세 조회
```
GET /api/investor/holdings/:holdingId
```

**인증 필요**: 예

**경로 파라미터**:

| 파라미터 | 타입 | 필수 | 설명 |
|---|---|---|---|
| `holdingId` | UUID | 필수 | 보유 기록 ID |

**응답** (성공 - 200):
```json
{
  "data": {
    "id": "holding_uuid",
    "channelId": "channel_uuid",
    "channelName": "뜬뜬 DdeunDdeun",
    "quantity": 10,
    "purchasePrice": 2500,
    "currentPrice": 2750,
    "currentValue": 27500,
    "gainLoss": 2500,
    "gainLossPercent": 10,
    "dividends": [
      {
        "date": "2026-04-01T00:00:00Z",
        "amount": 500
      },
      {
        "date": "2026-05-01T00:00:00Z",
        "amount": 500
      }
    ],
    "accumulatedDividends": 1500,
    "purchasedAt": "2026-03-15T00:00:00Z"
  }
}
```

**에러 응답**:

| 코드 | 상황 |
|---|---|
| 401 | 미인증 |
| 403 | 다른 사용자의 자산 조회 시도 |
| 404 | 보유 기록 없음 |

## 비즈니스 로직

### 포트폴리오 요약 계산
- **총 자산** = 모든 보유 자산의 현재 가치 합계
- **총 배당금** = 모든 보유 자산의 누적 배당금 합계
- **총 보유 조각 수** = 모든 채널의 보유 조각 수 합계
- **투자 중인 채널 수** = 보유 자산이 있는 채널의 개수

### 수익/손실 계산
- **gainLoss** = currentValue - purchasePrice
- **gainLossPercent** = (gainLoss / purchasePrice) × 100

### 배당금 지급
- 배당금은 채널의 배당 정책에 따라 정해진 주기(월간, 분기별 등)로 지급
- 누적 배당금에 누적
